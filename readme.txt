A collection of Gutenberg blocks for Minecraft.

== Description ==

MC Tools provides a collection of Gutenberg blocks designed for Minecraft-related WordPress sites.

== Installation ==

1. Upload the plugin files to `/wp-content/plugins/mc-tools/`
2. Activate the plugin through the "Plugins" screen in WordPress
